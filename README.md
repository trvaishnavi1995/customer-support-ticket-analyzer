# 📊 Customer Support Ticket Analyzer

## 📌 Overview
This project analyzes customer support tickets to identify patterns in customer issues, sentiment, and service quality.

## 🔧 Features
- Text cleaning (punctuation removal, normalization)
- Keyword-based analysis
- Priority distribution analysis
- Longest issue detection
- Unique word extraction

## 🧠 Skills Used
- Python
- Data Cleaning
- String Processing
- Logical Analysis

## 📈 Key Insights
- Frequent use of negative terms like "poor" and "slow"
- High-priority tickets highlight urgent issues
- Diverse customer language patterns

## 🚀 Future Improvements
- Sentiment analysis
- Dashboard visualization
- Real-time data integration
